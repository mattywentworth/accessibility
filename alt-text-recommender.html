Alt Text Recommender

Ensure your images have high-quality alt text for screen reader users

Upload a photo, then either write your own alt text or use the assistant to create or check it. Good alt text helps people using screen readers understand what the image shows.

Alt text should be a concise, accurate description that conveys what a sighted person would get from the image. You can type your own description, or you can click Submit Post with an empty field to have an AI-generated description suggested from the photo alone.

Or you can click Submit Post after uploading a photo and have an AI-generated description created for you. If you already added alt text, Submit will score how well it matches the image (1–10) and may suggest improvements.

Alt text is optional before submit. After upload, you can submit with an empty field to generate a suggestion, or add text to have it scored against the image.
